### Lessons Learned Today
1. **openclaw.json custom tool registration doesn't work** — agents don't see custom tools via this mechanism; must use exec with absolute paths in SKILL.md
2. **Session clearing mid-run loses work** — don't clear sessions while agent is actively processing; let it finish or steer it
3. **Stale .py scripts in listing folders cause confusion** — deleted old `generate_pdf.py` from 272CV folder; generate_listing_pdf.py now lives centrally
4. **Skills written for one interaction model (Slack) will silently fail on another (Telegram)** — always verify the interaction model matches the deployment
5. **Dynamic brand scanning** is essential — hardcoded list breaks every time a new brand is added (broke on Ulysse Nardin today)
6. **Prompt the agent with exact folder paths** when resuming — saves it from scanning and avoids picking wrong folder
