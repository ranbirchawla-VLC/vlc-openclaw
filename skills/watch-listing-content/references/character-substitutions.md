# Character Substitutions — Facebook Algorithm Avoidance
## For All Facebook Listings (Retail and Wholesale)

**Purpose:** Facebook's algorithm flags watch-related sales posts containing brand
names, payment terms, and certain commercial keywords. Character substitutions bypass
these flags while remaining readable to buyers. Apply these substitutions to the Key
Details line, description paragraphs, and payment block.

**Where to apply:** All Facebook group listings (retail and wholesale), including
Watch Trader Community, TWAT groups, Bay Watch groups, Moda Watch Club, and all
brand-specific Facebook groups.

**Where NOT to apply:** eBay, Chrono24, Grailzee, Reddit r/watchexchange, WTA
Dealer Chat, Value Your Watch, Instagram. These platforms do not have algorithm
filtering that requires substitutions. Use clean, unsubstituted text.

## Payment Block (Facebook Only)

Always use this exact format at the bottom of every Facebook listing:

```
W!re or Z3lle preferred (under $5K). USDT (crypto) and CC (+4.5% f33) available.
Ships fast from Colorado.
```

**CC fee is 4.5%.** This was updated March 2026 and is final. Do not use 3.5%.

**Location is Colorado.** Do not use DC or any other location.

---

## Application Rules

1. Apply substitutions to ALL text in Facebook listings: titles, Key Details pipe
   line, description paragraphs, payment block, and any brand/model references
2. Maintain readability. Substitutions should be obvious to human readers
3. When a new brand or term triggers Facebook suppression, add it to this table
4. If unsure whether a term needs substitution, apply it. False positives (unnecessary
   substitutions) are harmless. Missed substitutions can kill post visibility
5. The CTA "DM if you're interested" does not require substitutions
6. Emoji (🔎 before Key Details) does not require substitution
